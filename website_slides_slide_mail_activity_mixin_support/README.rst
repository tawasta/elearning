======================================
eLearning: Activity Support for Slides
======================================

* Adds 'Schedule an Activity' support for eLearning slides

Configuration
=============
* None needed

Usage
=====
* Open an individual slide in backend. The chatter section now
  allows scheduling activities related to the slide.
